Audio Visual elements

Audio

  • Autoplay: Boolean, use sparingly
  • Controls: provide playback cotrols for the user
  • loop: boolean, will play audio over and over
  • muted: boolean, use to mute the audio on the page
  • src: link to audio src file
  • volume: range 0.0 (silent) to 1.0 (loudest)

Video

  • autoplay: boolean, use sparingly
  • controls: provide playback controls
  • height: must be an absolute value, no percentage
  • muted: boolean, use this to mute the page on load
  • posted: use an alternative image in place of the first frame before playback has initialised
  • src: select the file you want
  • volume: range 0.0 to 1
  • width: may be an absolute value or a percentage
